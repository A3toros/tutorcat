# Character builder assets

Layered PNG/GIF character parts from [avatar_maker](https://github.com/oliveira-victor/avatar_maker) by Victor Oliveira / VFO Studio.

Tutorcat uses the original categories and layer order:

`body → skin → mouth → eyes → hair → features → shoes → bottom → top → extra`

**Refresh assets** (after cloning avatar_maker into `.tmp/avatar_maker`):

```bash
npm run sync:character-assets
```

Replace or re-license art for production if needed.
