
TutorCat is an educational language learning platform developed by students from  Mathayomwatsing School  for educational purposes. The project showcases modern web development practices and provides an engaging way for students to learn English.

What makes TutorCat special?

We made learning English feel natural: 
- Interactive speaking practice with real-time feedback
- Drag-and-drop vocabulary matching games
- Sentence construction exercises
- Grammar tips and corrections
- Earn stars and unlock achievements as you progress

It's actually fun: 
- Animated cat mascot that responds to your progress
- Multiple themes to personalize your experience
- Daily streak tracking to build learning habits
- Achievement system with fun titles from "Tiny Whisker" to "TutorCat"

Built for real learning: 
- Comprehensive lesson structure covering all language skills
- Progress tracking and personalized recommendations
- CEFR-aligned curriculum from A1 to C2 levels

Tech Stack
Frontend Framework & Core
-  Next.js 16.1.1  - React framework for server-side rendering, routing, and API routes
-  React 19.0.1  - UI library for building component-based interfaces
-  TypeScript 5.0.0  - Type-safe JavaScript for better code quality and developer experience
Styling & UI
- Tailwind CSS 3.3.0  - Utility-first CSS framework for rapid UI development
  - Used for responsive layouts, color schemes, and typography
  - Ensures consistent design across all pages and components
-  Lucide React  - Icon library for consistent iconography throughout the platform

   Animations & Interactions
-  Framer Motion 11.0.0  - Animation library for smooth page transitions and component animations
  - Powers the mascot animations and page transitions
  - Creates engaging micro-interactions throughout the UI
-  Konva 10.0.12  &  React Konva 19.0.0  - Canvas library for drag-and-drop interactions
  - Used for vocabulary matching games and grammar sentence construction
  - Enables interactive word dragging and placement
-  Lottie React 2.4.1  - Renders JSON-based animations for the cat mascot
  - Provides smooth, lightweight animations that respond to user actions

   Backend & Database
-  Netlify Functions  - Serverless functions for API endpoints
  - Handles authentication, lesson data, user progress, and activity submissions
  - Provides RESTful API architecture without managing servers
-  PostgreSQL (Neon)  - Cloud-hosted relational database
  - Stores user data, lessons, activities, achievements, and progress
  - Uses Neon DB for efficient database connections

   Authentication & Security
-  JWT (jsonwebtoken)  - Token-based authentication system
  - Secure user sessions and API access
  - HTTP-only cookies for enhanced security
-  Bcrypt.js  - Password hashing for secure credential storage

   Internationalization
-  i18next & React i18next  - Multi-language support
  - Enables English and Thai language interfaces
  - Dynamic language switching throughout the platform

   Data Visualization
-  Recharts 3.6.0  - Charting library for progress visualization
  - Displays user progress, statistics, and achievement tracking
  - Creates interactive charts on the dashboard

   Email Services
-  Resend  - Email delivery service
  - Sends OTP verification codes for user registration
  - Handles email-based authentication

  How the lessons work

 Every lesson takes you through 6 activities: 

1.  Quick chat  - Speaking warm-up to practice conversation
2.  New words  - Learn English vocabulary with Thai translations
3.  Word puzzles  - Drag-and-drop matching games
4.  Sentence building  - Construct sentences by arranging words
5.  Speaking practice  - Record speech and receive feedback
6.  Reading time  - Read passages and improve comprehension

 Progress tracking: 
- Earn stars for completing activities
- Unlock achievements and titles
- Track daily learning streaks
- Progress through CEFR levels from A1 to C2

  Features

-  User Dashboard  - View progress, achievements, and recommended lessons
-  Admin Panel  - Manage lessons, users, and platform statistics
-  Evaluation Test  - Assess English level and get personalized recommendations
-  Achievement System  - Gamified learning with 100+ achievements
-  Responsive Design  - Works seamlessly on desktop, tablet, and mobile devices

Languages
